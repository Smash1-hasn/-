# 爱即守护 — AI 语音关怀小程序

基于微信小程序的居家语音关怀平台。ASR Pro 离线语音识别 + ESP32 WiFi 中转 + DeepSeek 大模型 + 百度语音合成。

**一句话**：老人喊一嗓子，家人小程序收到告警。AI 陪聊陪老人说话，全程语音，不用打字。

---

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | 微信原生 WXML + WXSS + JavaScript ES6 |
| 后端 | 微信云开发（云函数 Node.js + 云数据库 + 云存储）|
| AI | DeepSeek Chat API（3 个独立场景：陪聊/问答/日报）|
| 语音识别 | 百度 STT API |
| 语音合成 | 百度 TTS API |
| 离线语音 | ASR Pro 2.0（天问 Block）|
| 包体积 | ~300KB，零第三方 UI 库 |

## 功能

| 功能 | 说明 |
|------|------|
| 🎤 语音 SOS | 喊"救命"自动告警，云端+本地双通道 |
| 🤖 AI 陪聊 | 按住说话 → 百度识别 → DeepSeek AI → 语音回复 |
| 📊 AI 日报 | 每日自动统计语音/告警 → DeepSeek 生成摘要 |
| 💬 双向留言 | 家人打字 ↔ 老人语音，自动转文字 |
| 💊 用药提醒 | 定时推送 + 历史联想输入 + 已服标记 |
| 🧩 健脑游戏 | 速算挑战 + 物品分类 |
| 🌤 天气提醒 | wttr.in 免费 API 自动切换安全提示 |
| 🏠 家庭配对 | 云端持久化配对码，换设备不丢 |

## 项目结构

```
miniprogram/      微信小程序（12 页面 + 4 云函数）
firmware/         ESP32 固件
docs/             设计文档 / 答辩准备（在 competition 分支）
```

## 快速开始

1. 微信开发者工具导入 `miniprogram/`
2. 开通云开发，部署 `cloudfunctions/` 下的 4 个云函数
3. 修改 `pages/elderly/chat/chat.js` 中的 DeepSeek 和百度 API Key
4. 编译预览
