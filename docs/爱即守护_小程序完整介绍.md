## 一、产品定位

"爱即守护"是一款基于微信小程序的居家老人语音关怀平台。产品通过 ASR Pro 离线语音模块与 ESP32 主控，实现老人语音关键词识别与云端通信。老人无需操作手机，喊出预设关键词即可触发紧急求救或传达日常信息给家人。家人通过小程序实时查看老人状态、接收告警和语音留言。

**一句话**：老人喊一嗓子，家人小程序收到。AI 陪聊陪老人说话，不用打字。

**核心用户**：
- 第一用户：70 岁以上居家老人（不识字、看不清屏幕、不会操作智能手机）
- 第二用户：异地工作的子女（想了解父母状态但无法全天候陪护）

## 三、语音交互系统（核心创新）

### 3.1 硬件侧

```
老人说话 → ASR Pro 离线识别 → 匹配关键词 → 串口发文字给 ESP32
                                                │
                             ┌──────────────────┤
                             │ SOS 关键词          │ 普通关键词
                             ▼                    ▼
                    云端 voice_sos 告警     云端 voice_msg 留言
                    → 家人告警页红色      → 家人留言板橙色
```

| 关键词 | ASR Pro 喇叭回复 | 云端类型 | 家人端看到 |
|--------|----------------|---------|----------|
| `救命` `我摔倒了` `救我` `快来帮我` | "已通知家人，请保持冷静" | voice_sos | 🔊 红色语音求救告警 |
| `我很好` `想你们了` `该吃药了` | "好的，已告诉家人" | voice_msg | 🔊 橙色语音留言 |

**离线优势**：ASR Pro 语音识别完全本地运行，不需要联网。断网情况下依然能识别关键词并通过 ESP32 的 WiFi 上传（前提是 WiFi 正常）。即使 WiFi 故障，ASR Pro 本身仍能播报回复安抚老人。

### 3.2 小程序侧

- 老人端首页设备状态卡片：5 分钟内语音确认"已发送给家人 ✓"绿色提示
- 家人端留言板：语音消息 🔊 图标 + 橙色左边框区分
- 家人端告警页：voice_sos 类型专属 🔊 图标
- 家人端仪表盘 AI 日报：统计当天语音消息数和告警数

## 四、用户体系设计

### 4.1 家庭配对系统

```
家人端                          老人端
  │                               │
  │ 登录 → 生成永久 FAM 码         │ 登录 → 输入 FAM 码
  │ 存入云数据库 users 集合        │ 存入云数据库 + 本地 Storage
  │                               │
  │ ←── 同一个 familyGroupId ──→  │
  │                               │
  │ 多个家人可共用同一 FAM 码       │ 多个老人可共用同一 FAM 码
  │ 数据按 familyGroupId 隔离      │ 每个老人由 _openid 区分
```

**配对码永久化**：
- 家人端首次登录生成永久码，存云端，换设备/清缓存不变
- 老人端首次输入后存云端 + 本地，清缓存后自动从云端恢复，无需重输
- 空码不可跳过，至少 4 位

### 4.2 角色双模式

| 维度 | 老人端 | 家人端 |
|------|--------|--------|
| 导航 | 单页滚动，无 Tab | 4 Tab 导航 |
| 字体 | 超大 48-68rpx | 标准 24-28rpx |
| 配色 | 暖金色调 | 深蓝科技 |

## 五、技术架构

### 5.1 总体架构

```
┌──────────────────────────────────────────────────────┐
│  硬件层                                                │
│  ASR Pro (离线语音) → ESP32 (WiFi+串口中转)            │
├──────────────────────────────────────────────────────┤
│  云端层 (微信云开发 cloud2)                             │
│  ├ 云数据库: 8 个集合                                  │
│  ├ 云函数: deviceDataIngest / aiHealthAnalysis         │
│  └ AI: DeepSeek API + 百度 STT/TTS                    │
├──────────────────────────────────────────────────────┤
│  小程序层 (原生 WXML+WXSS+JS)                          │
│  ├ 老人端 3 页 + 家人端 9 页 + 登录页 + 配对页        │

```

### 11.1 语音SOS全链路

**ESP32 固件（__WiFi_____.ino）**
```cpp
// 爱即守护 - ESP32 固件 v3.3（支持多命令词）
// 功能: 读取ASRPRO语音识别结果 → 根据内容判断类型 → 上传到云端

#include <WiFi.h>
#include <HTTPClient.h>
#include <HardwareSerial.h>

// ==================== 配置区域 ====================
#define WIFI_SSID       "su"
#define WIFI_PASSWORD   "subowen123"
#define DEVICE_ID       "001"
#define CLOUD_FUNC_URL  "https://cloud2-d2g7yeednd8ac1836-1433086696.ap-shanghai.app.tcloudbase.com/deviceDataIngest"

#define ASR_RX_PIN      16      // ASRPRO TX -> ESP32 GPIO16
#define REPEAT_INTERVAL 10000   // 防重复上传间隔(ms)

HardwareSerial asrSerial(2);
String lastText = "";
unsigned long lastUploadTime = 0;
unsigned long lastReconnectAttempt = 0;

// ==================== 判断是否为紧急求救 ====================
bool isEmergency(String text) {
    // 严格按照 ASRPRO 中的命令词判断
    return (text == "救命") || (text == "我摔倒了");
}

// ==================== 上传到云端 ====================
void uploadToCloud(String text, bool emergency) {
    if (WiFi.status() != WL_CONNECTED) {
        Serial.println("[WiFi] 未连接，无法上传");
        return;
    }
    
    String type = emergency ? "voice_sos" : "voice_msg";
    
    String json = "{";
    json += "\"deviceId\":\"" + String(DEVICE_ID) + "\",";
    json += "\"type\":\"" + type + "\",";
    json += "\"text\":\"" + text + "\",";
    json += "\"timestamp\":" + String(millis());
    json += "}";
    
    HTTPClient http;
    http.begin(CLOUD_FUNC_URL);
    http.addHeader("Content-Type", "application/json");
    
    int httpCode = http.POST(json);
    if (httpCode == 200) {
        Serial.println("[HTTP] 上传成功");
    } else {
        Serial.printf("[HTTP] 上传失败，错误码: %d\n", httpCode);
    }
    http.end();
}

// ==================== 初始化 ====================
void setup() {
    Serial.begin(115200);
    delay(1000);
    Serial.println("\n=== 爱即守护 v3.3 ===");
    
    asrSerial.begin(115200, SERIAL_8N1, ASR_RX_PIN, -1);
    Serial.println("[ASR] 串口已初始化");
    
    WiFi.begin(WIFI_SSID, WIFI_PASSWORD);
    Serial.print("[WiFi] 正在连接");
    int tries = 0;
    while (WiFi.status() != WL_CONNECTED && tries < 30) {
        delay(500);
        Serial.print(".");
        tries++;
    }
    if (WiFi.status() == WL_CONNECTED) {
        Serial.println("\n[WiFi] 连接成功，IP: " + WiFi.localIP().toString());
    } else {
        Serial.println("\n[WiFi] 连接失败，请检查配置");
    }
}

// ==================== 主循环 ====================
void loop() {
    static String buffer = "";
    while (asrSerial.available()) {
        char c = asrSerial.read();
        if (c == '\n' || c == '\r') {
            if (buffer.length() > 0) {
                String currentText = buffer;
                buffer = "";
                Serial.println("[ASR] " + currentText);
                
                unsigned long now = millis();
                if (currentText != lastText || (now - lastUploadTime) > REPEAT_INTERVAL) {
                    lastText = currentText;
                    lastUploadTime = now;
                    
                    bool emergency = isEmergency(currentText);
                    uploadToCloud(currentText, emergency);
                    
                    if (emergency) {
                        Serial.println("!!! 紧急情况，已上传 SOS !!!");
                    }
                } else {
                    Serial.println("[跳过] 重复内容或过于频繁");
                }
            }
        } else {
            buffer += c;
            if (buffer.length() > 200) buffer = "";
        }
    }
    
    if (WiFi.status() != WL_CONNECTED) {
        unsigned long now = millis();
        if (now - lastReconnectAttempt > 10000) {
            lastReconnectAttempt = now;
            Serial.println("[WiFi] 断线，尝试重连...");
            WiFi.reconnect();
        }
    }
    delay(10);
}
```

  const map = { help:'救命', fell:'我摔倒了', ok:'我很好', miss:'想你们了', pill:'该吃药了' }
  const text = map[rawText] || rawText
  const ts = new Date()

  try {
    const did = deviceId || '001'

    const dev = await db.collection('devices').where({ deviceId: did }).get()
    if (dev.data.length > 0) {
      await db.collection('devices').doc(dev.data[0]._id).update({ data: { status: 'online', lastSeen: ts } })
    } else {
      await db.collection('devices').add({ data: { deviceId: did, status: 'online', lastSeen: ts } })
    }

    const isSOS = type === 'voice_sos' ||
      (text && (text == 'help' || text == 'fell' ||
                text.indexOf('救命') >= 0 || text.indexOf('摔倒') >= 0 ||
                text.indexOf('救我') >= 0));

    if (isSOS && text) {
      const r = await db.collection('alerts').add({
        data: { deviceId: did, type: 'voice_sos', level: 'critical', message: '语音求救: ' + text, handled: false, timestamp: ts }
      })
      console.log('ALERT WRITTEN:', r._id)
    }

    if (!isSOS && text) {
      const r = await db.collection('messages').add({
        data: { type: 'voice', content: text, fromRole: 'elderly', fromUser: '老人(语音)', createTime: ts }
      })
      console.log('MSG WRITTEN:', r._id)
    }

    return { success: true, type: type || 'unknown' }
  } catch (err) {
    console.error('ERROR:', err.message)
    return { success: false, error: err.message }
  }
}

        },
