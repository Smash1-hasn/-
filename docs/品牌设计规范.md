# 安心守护 — 品牌设计规范

> 融合「Ocean Depths 可信赖感」+「Botanical Garden 温暖感」
> 主题名: **温暖守护 Warm Guardian**

## 字体层级（小程序适配）

微信小程序默认使用系统字体，我们按以下优先级：

- **大数字/关键数据**: `font-weight: 800` — 突出心率等核心数值
- **页面标题**: `font-weight: 700; font-size: 36-40rpx`
- **卡片标题**: `font-weight: 600; font-size: 28-32rpx`
- **正文**: `font-weight: 400; font-size: 26-28rpx`
- **辅助文字**: `font-weight: 400; font-size: 22-24rpx`

---

## 设计原则

1. **温暖的科技感** — 蓝白主调 + 珊瑚色点缀，不冰冷
2. **适老化**（老人端）— 超大字体 40rpx+，高对比度，大按钮 320rpx
3. **清晰层级** — 核心数据放最大，辅助信息退后
4. **一致的圆角** — 卡片 20-24rpx，按钮 48rpx，统一圆润感
5. **微动效** — fade-in、pulse 心跳、scale 点击反馈
