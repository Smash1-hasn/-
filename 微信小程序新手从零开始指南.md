# 微信小程序从零开始指南

> 写给零基础队友。跟着做就能把项目跑起来。

---

## 一、准备工作（30分钟）

### 1.1 注册小程序账号

1. 打开 https://mp.weixin.qq.com
2. 点右上角「立即注册」→ 选「小程序」
3. 用没注册过公众号的邮箱注册
4. 填写主体信息（个人即可，身份证+手机号+微信扫码）
5. 注册成功后，记下你的 **AppID**（设置→开发设置→AppID）

> 一个身份证只能注册 5 个小程序。

### 1.2 下载开发者工具

1. 打开 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载 **稳定版**（Windows/Mac）
3. 安装，用微信扫码登录

### 1.3 开通云开发（可选但推荐）

1. 开发者工具 → 顶部菜单「云开发」→ 点开通
2. 创建环境，名称随意，选免费版
3. 记下**环境ID**（如 `xxx-abc123`）

## 二、创建第一个小程序（10分钟）

### 2.1 新建项目

1. 打开微信开发者工具 → 点 **「+」** 新建项目
2. 填入：
   - 项目名称：随便写
   - 目录：选一个空文件夹
   - AppID：填你注册获得的 AppID
   - 开发模式：小程序
   - 模板：选「不使用模板」
3. 点确定

### 2.2 认识目录结构

```
miniprogram/
├── app.js          ← 小程序入口（全局逻辑）
├── app.json        ← 全局配置（页面列表、导航栏颜色）
├── app.wxss        ← 全局样式（所有页面通用）
│
├── pages/          ← 页面文件夹
│   └── index/
│       ├── index.js    ← 页面逻辑
│       ├── index.wxml  ← 页面结构（相当于HTML）
│       ├── index.wxss  ← 页面样式（相当于CSS）
│       └── index.json  ← 页面配置（标题等）
│
├── project.config.json  ← 工具配置（不用管）
└── cloudfunctions/      ← 云函数（如果开通了云开发）
```

### 2.3 四个文件干什么的

| 文件 | 相当于 | 写什么 |
|------|--------|--------|
| `.wxml` | HTML | 页面结构：`<view>` `<text>` `<button>` 等标签 |
| `.wxss` | CSS | 样式：颜色、大小、间距、动画 |
| `.js` | JavaScript | 逻辑：按钮点击做什么、数据怎么获取 |
| `.json` | JSON | 配置：页面标题、是否下拉刷新 |

## 三、基础开发（核心概念）

### 3.1 WXML 语法

```xml
<!-- 数据绑定：{{ }} 里面是变量 -->
<text>{{name}}</text>

<!-- 条件渲染 -->
<view wx:if="{{show}}">显示</view>
<view wx:else>隐藏</view>

<!-- 列表渲染 -->
<view wx:for="{{list}}" wx:key="id">
  <text>{{item.name}}</text>
</view>

<!-- 事件绑定 -->
<button bindtap="handleClick">点我</button>
<input bindinput="onInput" value="{{text}}" />
```

### 3.2 JS 写法

```javascript
Page({
  // 数据
  data: {
    name: '张三',
    list: [],
    show: true,
    text: ''
  },

  // 页面加载时执行
  onLoad() {
    console.log('页面加载了')
  },

  // 页面显示时执行（每次切回来）
  onShow() {
    // 刷新数据
  },

  // 按钮点击
  handleClick() {
    // 修改数据必须用 setData
    this.setData({ name: '李四' })
  },

  // 输入框输入
  onInput(e) {
    this.setData({ text: e.detail.value })
  }
})
```

### 3.3 WXSS 写法

```css
/* 选择器跟CSS一样 */
.title {
  font-size: 32rpx;       /* rpx = 响应式像素，会自动适配不同屏幕 */
  color: #333;
  padding: 20rpx;
}

.btn {
  background: #2B6CB0;
  color: #fff;
  border-radius: 10rpx;
}
```

### 3.4 页面跳转

```javascript
// 跳转到新页面（有返回按钮）
wx.navigateTo({ url: '/pages/detail/detail?id=1' })

// 切换 Tab 页
wx.switchTab({ url: '/pages/home/home' })

// 关闭所有页面跳到新页面
wx.reLaunch({ url: '/pages/login/login' })

// 返回上一页
wx.navigateBack()
```

### 3.5 常用 API

```javascript
// 弹窗
wx.showToast({ title: '成功', icon: 'success' })
wx.showModal({ title: '确认', content: '确定删除吗？', success(res) { if(res.confirm) { /* 删除 */ } } })
wx.showLoading({ title: '加载中...' })
wx.hideLoading()

// 存储
wx.setStorageSync('key', 'value')     // 存
wx.getStorageSync('key')              // 取
wx.removeStorageSync('key')           // 删
wx.clearStorageSync()                 // 清空

// 网络请求
wx.request({
  url: 'https://api.example.com/data',
  success(res) { console.log(res.data) },
  fail(err) { console.log('失败', err) }
})

// 拨打电话
wx.makePhoneCall({ phoneNumber: '13800138000' })

// 震动
wx.vibrateShort()   // 短震动
wx.vibrateLong()    // 长震动
```

## 四、添加新页面

### 4.1 创建页面文件

在 `pages/` 下新建文件夹 `detail/`，里面创建四个文件：
```
pages/detail/
├── detail.js
├── detail.json
├── detail.wxml
└── detail.wxss
```

每个文件最少写什么：

**detail.js**
```javascript
Page({
  data: {},
  onLoad() {}
})
```

**detail.json**（页面配置）
```json
{
  "navigationBarTitleText": "详情页"
}
```

**detail.wxml**
```xml
<view>
  <text>Hello World</text>
</view>
```

**detail.wxss**（可以空着）

### 4.2 注册页面

打开 `app.json`，在 `pages` 数组里加上：
```json
{
  "pages": [
    "pages/index/index",
    "pages/detail/detail"
  ]
}
```

**数组第一个是首页**，启动时默认打开。

## 五、云开发入门

### 5.1 初始化

在 `app.js` 里加：
```javascript
App({
  onLaunch() {
    wx.cloud.init({
      env: '你的环境ID'   // 开通云开发后获得的
    })
  }
})
```

### 5.2 操作数据库

```javascript
const db = wx.cloud.database()

// 查
db.collection('集合名')
  .where({ status: 'active' })    // 条件（可省略）
  .orderBy('time', 'desc')        // 排序（可省略）
  .limit(20)                       // 限制条数（可省略）
  .get()
  .then(res => { console.log(res.data) })
  .catch(err => { console.error(err) })

// 增
db.collection('集合名').add({
  data: { name: '张三', age: 25, time: new Date() }
}).then(res => { console.log('添加成功', res._id) })

// 改
db.collection('集合名').doc('记录的_id').update({
  data: { name: '李四' }
})

// 删
db.collection('集合名').doc('记录的_id').remove()
```

### 5.3 云函数

云函数让你在云端运行代码，可以调用外部 API。

**创建云函数**：
1. 右键 `cloudfunctions/` → 新建 Node.js 云函数 → 命名
2. 在 `index.js` 里写逻辑
3. 右键云函数文件夹 → 上传并部署：云端安装依赖

**云函数示例（index.js）**：
```javascript
const cloud = require('wx-server-sdk')
cloud.init({ env: cloud.DYNAMIC_CURRENT_ENV })

exports.main = async (event) => {
  const { name } = event   // 获取传入的参数
  return { message: `Hello ${name}` }
}
```

**小程序里调用**：
```javascript
wx.cloud.callFunction({
  name: '你的云函数名',
  data: { name: '张三' }
}).then(res => {
  console.log(res.result.message)  // Hello 张三
})
```

## 六、常见需求实现

### 6.1 Tab 导航栏

在 `app.json` 里配置：

```json
{
  "tabBar": {
    "list": [
      { "pagePath": "pages/index/index", "text": "首页" },
      { "pagePath": "pages/mine/mine", "text": "我的" }
    ]
  }
}
```

> Tab 数组最多 2-5 个，第一个默认展示。

### 6.2 下拉刷新

在页面 `.json` 里：
```json
{ "enablePullDownRefresh": true }
```

在页面 `.js` 里：
```javascript
Page({
  onPullDownRefresh() {
    // 刷新数据的逻辑
    this.loadData()
    wx.stopPullDownRefresh()  // 停止刷新动画
  }
})
```

### 6.3 分享

在页面 `.js` 里：
```javascript
Page({
  onShareAppMessage() {
    return {
      title: '分享标题',
      path: '/pages/index/index'
    }
  }
})
```

### 6.4 底部弹窗（ActionSheet）

```javascript
wx.showActionSheet({
  itemList: ['选项1', '选项2', '选项3'],
  success(res) {
    console.log('选中了', res.tapIndex)
  }
})
```

### 6.5 选择图片

```javascript
wx.chooseImage({
  count: 1,
  success(res) {
    const filePath = res.tempFilePaths[0]
    console.log('图片路径', filePath)
  }
})
```

## 八、上传与审核

### 8.1 上传代码

1. 开发者工具 → 右上角「上传」
2. 填版本号（如 1.0.0）和描述
3. 上传成功

### 8.2 提交审核

1. 打开 https://mp.weixin.qq.com
2. 左侧「版本管理」→ 找到刚上传的版本 → 点「提交审核」
3. 填写审核信息（功能页面、测试账号等）
4. 等待审核（1-7 个工作日）

### 8.3 发布

审核通过后 → 版本管理 → 点「发布」→ 用户就能搜到你的小程序了。

## 九、常见错误排查

| 错误 | 原因 | 解决 |
|------|------|------|
| `未找到xxx页面` | app.json 没注册这个页面 | 添加页面路径到 pages 数组 |
| `setData 报错` | 没在 Page() 里定义 data | 所有变量提前在 data 里声明 |
| 云数据库没数据 | 集合不存在 | 云开发控制台手动创建集合 |
| 云函数调用失败 | 没部署 | 右键云函数→上传并部署 |
| 图片不显示 | 路径不对 | 用绝对路径 `/images/xxx.png` |
| 模拟器白屏 | JS 报错 | 看底部控制台的红色报错信息 |

## 十、推荐学习顺序

1. **第一天**：注册账号 + 下载工具 + 创建第一个项目，改改文字看看效果
2. **第二天**：学数据绑定 `{{}}` 和事件 `bindtap`，做一个计数器按钮
3. **第三天**：学列表渲染 `wx:for`，做一个简单的待办列表
4. **第四天**：开通云开发，学会数据库增删查改
5. **第五天**：学页面跳转 `navigateTo`，把多个页面串起来
6. **第六天**：写你的项目，遇到不会的问我
